Research Vision:

 

My research interest focuses on real-world problems that require mobile robots (e.g., UAVs, driverless cars, planetary rovers, etc.) to traverse unknown obstacle-cluttered environments such as rugged planetary terrains, constrained regions, and urban scenarios. Here, perception, motion planning, and control modules play vital roles in a mission success. Robot dynamics limited onboard capacity in terms of computation and sensors, and environment features (e.g., obstacles, unknown rugged terrain, etc.) impose challenges for all modules and increase problem complexity which requires sophisticated and computationally expensive algorithms to solve the problem in hand. My research vision considers solving complex high-dimension mobile robotics problems in real-time with low computation power.

 

Research Interests: Motion planning and Control, reachability and traversability analysis, mobile robots (UAVs, car-like robots, planetary rovers), modelling and simulation.

 

Research Projects:

 

Continuous-Curvature (CC) Path Planning Methodology for Nonholonomic Vehicles

The work introduces and analyses Four parameter logistic (4PL) curve  as a  new  smooth path planning tool  that provides continuous  upper bounded  turn  rate command  for accurate  path  following. The proposed methodology analyses waypoints, curvature, and obstacle avoidance constraints in a 2-D design parameter space that provides multiple feasible solutions  for the problem  in-hand.  The thesis further considers:

·         Smooth path planning inside passages (prescribed boundaries) with guaranteed existence.

·         Collision-free maneuver generation for parallel parking a realistic-size car from any given starting position on a ray with a novel point-to-ray framework.

·         Generation of flyable UAV paths satisfying airspace restrictions such as, waypoint constraints, arbitrary shaped obstacle  avoidance,  and airspace  confinement.

In addition, the proposed tool is compared with the existing smooth path planning curves (such as Bézier curve, B-spline, and clothoid)  based solutions  to highlights  the significant computational advantages.

 

Fast and   Safe Motion Primitive Generation for Agile Quadrotor Flights:

This work considers the generation of agile kinodynamic trajectories using motion primitives with fast computation for online operations. Using the differential flatness property of the quadrotor, the concerned problem can be mapped in a lower dimension curve-fitting problem where the curve satisfies the feasibility constraints (flyability, obstacle avoidance, narrow gap traversal). In contrast to the existing state-of-art literature which considers a polynomial time-parameterized curve fitting approach, my work explores the possibility of using logistic curves to achieve further parameter reduction for the problem.  At this end, I have developed a trajectory generation for a narrow gap traversal application which can generates thousands of motion primitives in a fraction  of seconds and validated it with 6-DoF quadrotor models.

 

Safe Motion Planning for Planetary Hopping Robots:

Hopping robots can cover long distances in lesser time on rugged terrains, jump over large obstacles, and provide visual data from high altitudes to investigate diverse and disconnected regions which makes them a prospective candidate for planetary surface exploration application. The key challenges for hopping robot trajectory planning  are system and environment errors (localization, modelling and control),  uncontrolled flying phase,  and hybrid  automata of the robot  that cause undesirable robot  motions  (such as drifting,  bouncing,  sliding,  tumbling, and  rolling)  which lead  to  path  following errors  and  dangerous  landing of the  rover.   Addressing the issues, I have developed a safe Voronoi-based waypoint generator that provides a safe waypoint roadmap for a disconnected terrain scenario with forbidden zones.  I have further shown that the proposed waypoint generator can be integrated in a path planning framework for generating the robust hopping path between given start and goal.  Viability of the proposed approach is tested on a realistic planetary terrain scenario of Mars.

 

Robust Output Feedback Controller Design

The main emphasis of my master’s work was on designing of an observer-based feedback controller for uncertain systems with guaranteed stability and performance.  This work considers three output feedback controllers, namely, sliding mode observer-controller (SMO-SOSMC),  extended  state  observer  (ESO)-SOSMC, and  active  disturbance rejection  controller (ADRC),  for controlling a 1-DoF Vertical Take-Off Landing (VTOL)  system. With simulations and experiments it is demonstrated that the ADRC perform best  among  all three  in presence  of unknown  disturbance and  uncertainties due to its model independent attribute followed by ESO-SOSMC  and then  SMO-SOSMC.
